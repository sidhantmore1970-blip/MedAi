# 🧠 MedAI — AI-Powered Medical Assistant

MedAI is an AI-powered healthcare assistant designed to analyze prescriptions, detect critical warnings, and provide intelligent responses using advanced logic. It helps improve patient safety and supports healthcare decision-making.

---

## 🚀 Features

* 🧾 **Prescription Data Extraction**
  Extracts structured data from medical prescriptions.

* ⚠️ **Critical Warnings Detection**
  Identifies potential risks such as drug conflicts or unsafe prescriptions.

* 🤖 **AI Assistant Logic**
  Provides intelligent responses based on medical data.

* 🌐 **FastAPI Backend**
  High-performance API built using FastAPI.

* 📍 **Location-based Support**
  Includes basic location handling features.

---

## 🛠️ Tech Stack

* **Backend:** FastAPI
* **Database:** SQLite
* **Language:** Python 3.11+
* **Frontend:** HTML (Basic UI)

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/sidhantmore1970-blip/MedAi.git
cd MedAi
```

### 2. Create virtual environment (optional but recommended)

```
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Run the FastAPI server

```
uvicorn fast_api_file:app --reload
```

### 5. Open in browser

```
http://127.0.0.1:8000
```
